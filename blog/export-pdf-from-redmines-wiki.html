Export PDF from Redmine's Wiki

Sat 02 July 2011
By Yusaku ONO

In content.

I'll introduce a Redmine's plugin, DocPu. It's enable us to export PDF from Redmine's Wiki.

I tried to set up the plugin on Debian Squeeze.

Install LaTeX packages.

DocPu uses LaTeX environment, so we must install LaTeX packages.

I installed these packages using aptitude.

  • ptex-bin
  • ptex-buildsupport
  • ptex-jtex
  • texlive-latex-recommended
  • texlive-latex-extra
  • cjk-latex
  • latex-cjk-japanese
  • latex-cjk-japanese-wadalab

Install DocPu

Download and locate the plugin's directory to $RAILSROOT/vendor/plugins.

$ wget http://www.redmine.org/attachments/4975/redmine_doc_pu_v003.zip
$ unzip redmine_doc_pu_v003.zip
$ mv redmine_doc_pu $RAILSROOT/vendor/plugins/
$ sudo /etc/init.d/apache2 restart

After that, you should see the plugin's item in above menu [Administration] -> [Plugin].

Customize a template

Default templates doesn't include Japanese version.

I wrote a template based on artile_en_tcfi.tex (one of the default templates).

# diff article_ja_tcfi.tex article_en_tcfi.tex
21d20
< \usepackage{CJKutf8}
38d36
< \begin{CJK}{UTF8}{min}
54d51
< \end{CJK}
