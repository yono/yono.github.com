Publish the blog create by Pelican at github

土 11 6月 2011
By Yusaku ONO

In content. 

$ mkdir -p ~/.pelican/content
$ vim ~/.pelican/setting.py
$ cat ~/.pelican/setting.py
#!/usr/bin/env python

OUTPUT_PATH = '/path/to/github/site/repository/blog/'
$ vim ~/.pelican/content/hoge.md
$ pelican -s ~/.pelican/setting.py ~/.pelican/content
$ cd /path/to/github/site/repository/
$ git add .
$ git commit -a -m 'add blog by pelican'
$ git push origin master

It may be better to write a script for automation creating blog and deploying.
