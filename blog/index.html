Install Pelican with virtualenvwrapper

土 11 6月 2011
By Yusaku ONO

In content.

I installed Pelican as a environment to write a blog much easier.

% mkvirtualenv pelican
(pelican) % pip install pelican
(pelican) % mkdir content && cd content
(pelican) % vim hoge.md
(pelican) % pelican ~/content
